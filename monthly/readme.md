主要修改内容：
1. 支持年度和月度数据
新增 extract_time() 函数，支持多种时间格式

智能识别年度数据（YYYY）和月度数据（YYYYMM）

自动设置合适的时间坐标

2. 数据频率自动检测
自动判断数据是年度还是月度

在界面上显示数据频率信息

根据频率优化默认参数

3. STL 分解修复
修复了 STL 结果显示时的维度错误

直接使用返回的二维统计量，不再调用 .mean("time")

为年度数据添加警告提示

4. 改进的用户界面
显示数据频率和时间范围

根据数据类型提供分析建议

更清晰的参数说明

5. 向后兼容性
完全支持原有的年度数据命名方式

分析结果与原来一致

不影响现有用户的使用习惯

6. 文件命名示例
年度数据（继续使用）：

text
NDVI_2000.tif
NDVI_2001.tif
NDVI_2002.tif
月度数据（新增支持）：

text
NDVI_200001.tif  # 2000年1月
NDVI_200002.tif  # 2000年2月
NDVI_200012.tif  # 2000年12月
这个修改版本既保持了年度数据的完全兼容，又新增了对月度数据的支持，同时修复了之前发现的错误。
